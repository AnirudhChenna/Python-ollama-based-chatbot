# Python-ollama-based-chatbot
A fully local RAG chatbot powered by Ollama — no API keys, no per-token costs. Upload your documents and get grounded, cited answers using local LLMs (Llama 3.1, Mistral, Qwen2.5) with semantic search (BAAI/bge embeddings) and cross-encoder re-ranking for high-precision retrieval, all backed by a persistent ChromaDB vector store.
